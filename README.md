# Cartify

A modern, scalable E-commerce REST API built with **Node.js**, **Express.js**, and **MongoDB**. Cartify provides everything needed for an online store, including authentication, product management, shopping cart, orders, coupons, online payments, and an admin dashboard.

---

## 🚀 Features

### Authentication & Authorization

- User registration and login
- JWT Authentication
- Role-based authorization
- Password hashing using bcrypt
- Forgot password & password reset
- Email verification
- Paymob payment integration
- Payment Intention API
- Retry failed payments
- Secure payment workflow

### API Features

- RESTful API
- Authentication & Authorization
- Pagination
- Filtering
- Sorting
- Searching
- File Upload
- Validation
- Error Handling
